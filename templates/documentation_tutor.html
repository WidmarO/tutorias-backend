Method : PUT

This endpoint add a new tutor in the DB

required fields: cod_tutor,cod_teacher,cod_tutoring_program Example (request) : 
                        
                            Headers 
                            Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.....eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9
